Início Guia para Iniciantes Sobre Repositório

Makicore é rápido, lógico e educacional.

Um console tátil e uma linguagem de programação criados para estudantes.

O ecossistema Makicore oferece:

  • Hardware Tátil: Console baseado em engenharia de dobras estruturais.
  • Programação Maki: Programação simplificada para automação de problemas matemáticos.
  • Foco Escolar: Adequado para o Ensino Fundamental I e II.
import maki.console.*;

func iniciar_exemplo() {
  print("Isto é um exemplo...");
  definir objeto = "cubo";
}
